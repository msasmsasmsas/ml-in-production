# PR5: Hyperparameter Search with NNI

This directory contains code for conducting hyperparameter searches using Microsoft NNI.

## Files
- `nni_search.py`: Script for hyperparameter search with NNI.
- `config.yml`: NNI configuration file.
- `requirements.txt`: Dependencies.

## Usage
1. Install dependencies: `pip install -r requirements.txt`.
2. Start NNI experiment: `nnictl create --config config.yml`.

## Output
Results are available in the NNI experiment dashboard.
